## :running: Run locally

Clone the project

```bash
  git clone https://github.com/trstefan/react-image-slider.git
```

Go to the project directory

```bash
  cd react-image-slider
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
